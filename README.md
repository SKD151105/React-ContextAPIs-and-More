# React Context APIs and More

A focused React study repo for revision. The goal is to keep the content minimal, high-quality and relevant.

## Core path

1. Start with [Basics.jsx](00_Basics/Basics.jsx)
2. Learn how to think in React with [ThinkingInReact.jsx](02_Thinking_In_React/ThinkingInReact.jsx)
3. Study component structure in [ClassicTemplate.jsx](01_Classic_Template/ClassicTemplate.jsx)
4. Learn rendering, state, and events in [StateEventsAndRendering.jsx](02_State_Events_And_Rendering/StateEventsAndRendering.jsx)
5. Learn forms and data flow in [FormsAndDataFlow.jsx](03_Forms_And_Data_Flow/FormsAndDataFlow.jsx)
6. Learn routing in [RoutingAndAuthorization.jsx](04_Routing_And_Authorization/RoutingAndAuthorization.jsx)
7. Learn Context and custom hooks in [ContextAndCustomHooks.jsx](05_Context_Custom_Hooks/ContextAndCustomHooks.jsx)
8. Learn shared state with [ContextAPI.jsx](07_Context_API/ContextAPI.jsx)
9. Learn Redux basics in [ReduxToolkit.jsx](08_Redux_And_Toolkit/ReduxToolkit.jsx)
10. Learn effects and data fetching in [EffectsAndFetching.jsx](09_Effects_And_Data_Fetching/EffectsAndFetching.jsx)
11. Handle errors in [ErrorHandling.jsx](15_Error_Handling/ErrorHandling.jsx)
12. Learn accessibility basics in [AccessibilityAndForms.jsx](20_Accessibility_And_Forms/AccessibilityAndForms.jsx)
13. Learn testing basics in [TestingReactComponents.test.jsx](21_Testing_React_Components/TestingReactComponents.test.jsx)

## How to use it

- Learn the core path in order
- Read the examples before memorizing theory
- Focus on when to use each topic in real projects
- Skip the advanced folders unless you want extra depth later

## Source reference

The notes are grounded in the official React docs: [react.dev/learn](https://react.dev/learn)
